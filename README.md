# Daily Schedule Tracker 🐾

Persoonlijk dagelijks schema tracker gemaakt door Marc Brouwer.

## Wat doet dit programma?
- Toont de huidige datum en tijd
- Interactieve checklist voor dagelijkse medicijnen
- Checklist voor verzorging van huisdieren
- Dynamische vinkjes op basis van jouw antwoorden

## Waarom dit project?
Dit is mijn eerste Python project als onderdeel van mijn leerpad 
richting SOC Analyst via OptiSec.

## Hoe gebruik je het?
1. Zorg dat Python geïnstalleerd is
2. Clone de repository
3. Run in de terminal:
   python checklist.py

## Medicijnen
- Esomeprazol (nuchter, 1 uur voor ontbijt)
- Metformine + Gliclazide (bij ontbijt)
- Metformine + bipolaire medicatie (bij avondeten)
- Temazepam (optioneel, slaapondersteuning)
